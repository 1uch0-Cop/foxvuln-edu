# FoxVuln EDU 1.0.1

Versión de mantenimiento de FoxVuln EDU.

## Cambios

- Se eliminaron rutas personales de las instrucciones públicas.
- La configuración del modo docente utiliza la variable portable `$USER`.
- Se mejoró la documentación de instalación y ejecución.
- Se mantienen los controles de seguridad y la allowlist educativa.
- No se incorporan nuevas funciones de exploración.

## Verificación

- 19 pruebas automáticas aprobadas.
- Modo estudiante validado.
- Modo docente validado.
- Compatibilidad comprobada en Fedora y Kali Linux.
